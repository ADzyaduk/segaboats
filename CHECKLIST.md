# ✅ Чеклист для развертывания

## Перед установкой

- [ ] Арендован чистый Ubuntu сервер (20.04+ или 22.04+)
- [ ] Есть root доступ (sudo)
- [ ] Есть домен (или IP адрес для тестирования)
- [ ] Создан Telegram бот и получен токен

## После установки

- [ ] Docker и Docker Compose установлены
- [ ] Репозиторий клонирован в `/opt/boats2026`
- [ ] `.env` файл создан и заполнен
- [ ] Docker контейнеры запущены (`docker compose ps`)
- [ ] База данных создана (таблицы существуют)
- [ ] Системный Nginx установлен и запущен
- [ ] Сайт открывается по IP: `curl http://YOUR_IP`
- [ ] DNS настроен (A-запись указывает на IP сервера)
- [ ] Сайт открывается по домену: `http://your_domain`

## Проверка работы

- [ ] Главная страница загружается
- [ ] API работает: `curl http://your_domain/api/boats`
- [ ] Админ-панель доступна: `http://your_domain/admin`
- [ ] n8n доступен (если настроен доступ)
- [ ] Telegram бот отвечает (если настроен)

## Безопасность

- [ ] Пароли изменены (не используются значения по умолчанию)
- [ ] Firewall настроен (порты 22, 80, 443 открыты)
- [ ] Регулярные бэкапы БД настроены (опционально)
